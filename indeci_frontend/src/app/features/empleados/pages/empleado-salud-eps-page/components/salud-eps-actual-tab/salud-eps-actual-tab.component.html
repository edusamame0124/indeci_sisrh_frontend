Reglas clave para el cálculo

  1. Base aplicable = menor entre remuneración CAS y 45% de la UIT vigente.
  2. Si el trabajador tiene Solo EsSalud → Aporte empleador = Base × 9%.
  3. Si el trabajador tiene EsSalud + EPS →
    • Aporte EsSalud empleador = Base × 6.75%
    • Crédito EPS = Base × 2.25%
    • Total aporte = 9%
  4. La configuración de vigencias y tasas se gestiona en el módulo Configuración EsSalud / EPS.

Importante

  • Esta información NO genera descuentos al trabajador.
  • Si el trabajador tiene un descuento voluntario asociado a una EPS, registrarlo en el módulo de Conceptos.
  • Cambios en esta configuración afectan el cálculo de EsSalud del empleador en planillas futuras.
